# Souss Massa 360 🌄

**Discover the Authentic Berber Soul of Southern Morocco**

A platform connecting travelers with authentic Berber homestays and cultural experiences in the stunning Souss-Massa region of Morocco.

## 🌍 Moroccan Identity

This project celebrates the rich Berber (Amazigh) culture of Southern Morocco, featuring:

- **Berber Homestays**: Traditional homes in Paradise Valley, Taghazout, Taroudant, Tafraoute, and Agadir
- **Cultural Experiences**: Argan oil workshops, traditional cuisine (tagine, couscous), and artisan crafts
- **Language Support**: French, Arabic, and Tamazight (Berber language) options
- **Moroccan Cities**: Agadir, Marrakech, Taroudant, Tafraoute, Taghazout, and Paradise Valley

## 🚀 Features

- **Authentic Stays**: Connect with local Berber families
- **Cultural Immersion**: Learn traditional crafts and cuisine
- **Sustainable Tourism**: Support local communities
- **Multi-language**: French/Arabic/Tamazight support

## 🛠️ Tech Stack

**Frontend:**
- React 18 with TypeScript
- Vite for fast development
- Tailwind CSS for styling
- DaisyUI components

**Backend:**
- Node.js with Express
- TypeScript
- MongoDB with Mongoose
- JWT authentication

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- MongoDB (local or Atlas)

### Installation

1. **Clone and setup:**
   ```bash
   git clone <repository-url>
   cd souss-massa-360
   ```

2. **Backend setup:**
   ```bash
   cd backend
   npm install
   cp .env.example .env  # Configure environment variables
   npm run dev
   ```

3. **Frontend setup:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Open browser:**
   - Frontend: http://localhost:5173
   - Backend API: http://localhost:5000

## 📍 Moroccan Locations Featured

- **Agadir**: Coastal city and surfing destination
- **Taghazout**: Surfing paradise with Berber villages
- **Taroudant**: "Little Marrakech" with argan groves
- **Tafraoute**: Famous for painted rocks and desert landscapes
- **Paradise Valley**: Authentic Berber villages and waterfalls
- **Immouzer**: Mountain views and coastal charm

## 🎨 Berber Culture Highlights

- **Traditional Architecture**: Mud-brick homes and riads
- **Cuisine**: Tagine, couscous, pastilla, and Berber tea
- **Arts & Crafts**: Argan oil production, pottery, carpets
- **Hospitality**: Amazigh values of welcome and generosity
